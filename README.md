# TaskWebApp

## 📚 Overview
TaskWebApp is a task management application built using **ReactJS** with **Vite** for fast development and optimized performance. 

## 🚀 Features
- Create, update, and manage tasks.
- Lightweight and fast application using Vite.

## 🛠️ Technologies Used
- ReactJS
- Vite
- JavaScript
- CSS
